# Lost & Found System

This is a simple PHP-based lost and found system.

## Structure

- `index.php` landing / login page
- `login.php`, `register.php` for authentication
- `dashboard.php` user dashboard
- `report-lost.php`, `report-found.php` for submissions
- `search.php`, `claim.php` for item lookup/claims
- `admin/` for administrative functions
- `includes/` contains configuration and reusable snippets
- `assets/` for CSS, JS, and images

## Notes

- Use PDO for database interactions.
- Adapt folder names if you add more modules.
